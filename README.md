RiftApp combines **chat, voice, DMs and shared spaces** into one fast minimal experience.

## Quick start

### 1. Setup env

cp backend/.env.example backend/.env

### 2. Run

docker compose up -d

That’s it. Rift is running.

Full compose config is in `backend/compose.yml`.
